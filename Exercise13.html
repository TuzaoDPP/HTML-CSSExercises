What is HTML

HTML is one of the most popular markup languages ​​and uses a simple code format, which can be generated without the help of special applications.

To create an HTML document, all you need is an ASCII text editor like NotePad (Notepad), for example. If you need to check document formatting during creation, you can use any browser as they are all capable of interpreting the code.

Main commands

Commands are enclosed in code between less than "<" and greater than ">" signs and are called tags.
